Thanks for submitting your job!

Someone from the community will review your job and publish it as soon as possible in the jobs website

In the meantime, you can:

  • Read our code of conduct
  • See all the current jobs
  • Read more about Open Source Design

What actually happens (gory details)

Submitting a job sends a pull request to the Open Source Design jobs repository. Someone needs to merge the pull request. Merging triggers the publication to the Open Source design jobs website
